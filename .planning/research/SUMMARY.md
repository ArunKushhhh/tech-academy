# Research Summary

## Stack
- Use the existing Next.js, Tailwind CSS, and shadcn/ui stack.
- Highly recommended to use **Web3Forms** or **Resend** for serverless form submissions without a backend.
- Export as static HTML for optimal speed.

## Table Stakes
- Clear Hero Value Proposition
- Course Catalog & Philosphy/About Us
- Testimonials
- High-conversion Contact Form

## Watch Out For
- **Lost Form Submissions:** Must handle form API responses gracefully and verify emails are sent.
- **Animation Performance:** Ensure GSAP/Aurora components don't freeze mobile devices.
- **Navigation:** Deep-links (`#contact`) must use smooth scrolling.
