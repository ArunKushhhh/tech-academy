# Roadmap: Tech Academy

## Overview

A structured plan to transform the existing static Next.js skeleton into a fully functional, high-converting landing page for a single-educator academy.

## Phases

- [x] **Phase 1: Educator & Hero Experience** - Establish the visual hook and educator profile
- [x] **Phase 2: Catalog & Philosophy** - Build out the courses and academy context
- [ ] **Phase 3: Conversion & UX Optimization** - Implement contact form and finalize single-page navigation

## Phase Details

### Phase 1: Educator & Hero Experience
**Goal**: Establish the primary visual hook and introduce the educator immediately to visitors.
**Depends on**: Nothing
**Requirements**: CONV-01, INFO-01
**Success Criteria** (what must be TRUE):
  1. Hero section loads instantly with optimized visual assets.
  2. Educator Profile is visible immediately upon scrolling down.
**Plans**: TBD

Plans:
- [ ] 01-01: Build Educator Profile section and refine Hero messaging

### Phase 2: Catalog & Philosophy
**Goal**: Build out the core academy information and visually showcase the courses.
**Depends on**: Phase 1
**Requirements**: INFO-02, INFO-03, INFO-04
**Success Criteria** (what must be TRUE):
  1. Visitors can clearly read through available courses.
  2. Academy philosophy and testimonials render cleanly across mobile and desktop.
  3. All sections are seamlessly connected to the single-page layout.
**Plans**: TBD

Plans:
- [ ] 02-01: Develop interactive Course Catalog section
- [ ] 02-02: Implement Academy Philosophy and integrate Testimonials

### Phase 3: Conversion & UX Optimization
**Goal**: Hook up the contact form to capture leads and ensure technical performance.
**Depends on**: Phase 2
**Requirements**: CONV-02, CONV-03, UX-01, UX-02, UX-03
**Success Criteria** (what must be TRUE):
  1. Form submissions arrive in the educator's email inbox reliably.
  2. Sticky CTA smoothly scrolls users down to the contact form.
  3. Site passes performance checks and exports as static HTML successfully.
**Plans**: TBD

Plans:

- [x] 03-02: Implement smooth scrolling navigation and global CTA
