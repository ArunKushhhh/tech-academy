# Plan 03-02: Navigation Wiring, Section Anchors, CTA Buttons & Footer Links

## Objective
Wire up the entire single-page navigation: add `id` anchors to all sections, update navbar items config to real links, rename both "Join Now" CTA buttons to "Contact Us" → `#contact`, update footer links, and enable CSS smooth scrolling globally.

## Tasks

### Task 1: Add scroll-behavior: smooth to globals.css

<read_first>
- app/globals.css (current content — must not break existing @import or @theme blocks)
</read_first>

<action>
Add the following CSS rule to `app/globals.css`. Place it AFTER the `@theme inline { ... }` block and BEFORE any `@layer` blocks. This works correctly with Tailwind v4's `@import "tailwindcss"` approach:

```css
html {
  scroll-behavior: smooth;
}
```

Do NOT add inside `@theme`, `@layer`, or any at-rule block — it must be a top-level rule.
</action>

<acceptance_criteria>
- `app/globals.css` contains the exact text `scroll-behavior: smooth;`
- The rule is at top level (not nested inside any at-rule block)
</acceptance_criteria>
